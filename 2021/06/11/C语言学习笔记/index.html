基本的输入和输出函数的用法

printf的用法

printf() 输出 //重点
将变量的内容输出到显示器上

输出控制符包含如下

%d                  --  int
%ld                 --  long int
%c                  --  char
%f                  --  float
%lf                 --  double
%x(或者%#x或者%#X)   --  int 或 long int 或 short int
%o                  --  同上
%s                  --  字符串

所以推荐使用%#X,因为当输出2f或2F容易与其他数字或进制混淆

printf四种用法
    1. printf("字符串\n");
    2. printf("输出控制符",);
    3. printf("输出控制符1 输出控制符2....", 输出参数1, 输出参数2, ....); 输出控制符和输出参数的个数必须一一对应
    4. printf("输出控制符 非输出控制符", 输出参数);

为什么需要输出控制符

  1. 01组成的代码可以表示数据也可以表示指令
  2. 如果01组成的代码表示的是数据的话,那么同样的01代码组合以不同的格式输出就会有不同的输出结果
    所以需要输出控制符指定代码输出的格式

scanf的用法

scanf() : 通过键盘将数据输入到变量中
scanf() 输入 //java已经没有
两种用法:
用法一: scanf(“输入控制符”, 输入参数);
功能: 将从键盘输入的字符转化为输入控制符所规定格式的数据,然后存入以输入参数的值为地址的变量中
